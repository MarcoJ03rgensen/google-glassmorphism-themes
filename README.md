# Google Glassmorphism Themes

A Chrome extension that transforms Google Search with beautiful glassmorphism UI and ambient ocean animations.

## Features
- **Glass UI**: Translucent, blurred cards for search results.
- **Ambient Animations**: Background "God rays" and rising bubbles (inspired by deep sea diving).
- **Themes**: Switch between "Deep Ocean", "Apple Frost", and "Neon Night".

## Installation

1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked**.
5. Select the folder containing this repository.
6. Open Google (e.g., [google.com](https://google.com) or [google.dk](https://google.dk)) to see the changes.

## Usage

Click the extension icon in your browser toolbar to switch themes.
